* use main file in client folder to play the game
The project below is using raylib and boost.asio for multiplayer (still in development)

*build instruction
- use MSys32 MINGW64
- navigate to [projectFolder]/client or server
- run command line
    - mkdir build
    - cd build
    - rm -r *
    - cmake .. -G "MinGW Makefiles"
    - cmake --build .
    

use wasd to move,
k= attack,
press i when on shopping or arrow symbol to enter building,

run main.exe to play the game.
